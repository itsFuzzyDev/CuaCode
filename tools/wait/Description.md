Pauses for the given number of seconds (max 10). Use this to let a
UI settle after an action — e.g. after app_open, before a page
finishes loading, or after an animation.
> NOTE: Usually websites, apps, and all of the sorts are very fast to open, and don't forget that you're an AI, it takes time between loops that might already take the time you're trying to wait on with this tool, be weary.
